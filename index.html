My Work

Sudoku Solver

Created a Sudoku solver in Java utilizing backtracking, an algorithmic technique to effectively fill the grid. The program employs recursion and data structures to explore possible number placements, ensuring adherence to Sudoku rules and solving puzzles in a structured and efficient way. This project deepened my knowledge of DSA concepts, including recursion, backtracking, and solving grid-based problems.

Railway Reservation System

Developed a basic railway reservation system using HTML, CSS, and JavaScript. The website consists of four pages: a main booking page, a food ticket booking page, an about/help page, and a confirmation page. Users can book train tickets, select food options, and navigate easily through the user-friendly interface. This project helped me build foundational web development skills and understand the integration of front-end technologies in creating a functional, interactive system.
